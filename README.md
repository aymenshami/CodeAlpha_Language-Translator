
A simple language translator built using Googletrans and Gradio.
It allows you to translate text between different languages through an interactive web interface.

Features
Translate text from any language to any other language supported by Google Translate.
Interactive and easy to use Gradio interface.
Handles errors gracefully if translation fails.

Source Text: Hello, My name is Aiman
Source Language: English
